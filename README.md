# Cálculo de Salário com Horas Extras

## Descrição

Aplicação desenvolvida em Python para cálculo do salário mensal com base na carga horária trabalhada. O sistema implementa regras de negócio relacionadas à jornada padrão de trabalho e à remuneração de horas extras, considerando adicional percentual sobre o valor da hora regular.

A solução foi construída com foco em organização lógica, clareza de saída e aplicação prática de conceitos fundamentais de programação.

---

## Objetivo

Implementar um algoritmo capaz de:

* Receber dados de entrada (horas trabalhadas e valor da hora)
* Processar regras de cálculo salarial
* Retornar resultados de forma estruturada

---

## Regras de Negócio

* Limite de horas normais: 160 horas mensais
* Horas excedentes classificadas como horas extras
* Adicional de 50% aplicado sobre o valor da hora nas horas extras

---

## Funcionalidades

* Validação e processamento de entrada de dados
* Cálculo de horas normais e horas extras
* Aplicação de percentual de acréscimo
* Geração de saída detalhada no terminal
